## 📋 About AIR-J

AIR-J is a simple language designed for artificial intelligence programs to use, but it stays out of the way for humans. This version runs on the Java Virtual Machine (JVM), which means it works on many computers with Java installed. You do not need programming skills to use it.

This project is ideal if you want to explore AI languages or test AI-related code without dealing with complex setups.

Before you start, please make sure your computer meets these requirements:

## 📂 Installing and Running AIR-J

After downloading AIR-J, follow these steps to run it on Windows:

1. If the downloaded file is a `.zip`:
   - Right-click the file and choose **Extract All**.
   - Select a destination folder or accept the default.
   - Click **Extract** to unzip the files.

2. Make sure you have Java installed and working:
   - Open Command Prompt by typing `cmd` in the Start menu.
   - Type `java -version` and press Enter.
   - You should see version information for Java. If you get an error, install Java first.

3. Locate the AIR-J executable file. It will usually be a `.jar` file inside the extracted folder.

4. To run AIR-J:
   - Open Command Prompt.
   - Change directory to the folder where AIR-J is saved. Example:  
     `cd Desktop\AIR-J`  
   - Run AIR-J by typing:  
     `java -jar AIR-J.jar`  
   
5. AIR-J should now start running on your computer.
